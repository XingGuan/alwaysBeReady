CSS帧动画替代GIF方案

使用CSS动画和静态图片序列创建高性能动画效果

示例1: 旋转风扇

实现原理

/* 雪碧图动画关键帧 */
@keyframes fan {
    from { background-position: 0 0; }
    to { background-position: -2400px 0; }
}

.fan-animation {
    animation: fan 0.6s steps(7) infinite;
}

示例2: 跳动的心

❤️

实现原理

/* 变形动画关键帧 */
@keyframes heartbeat {
    0% { transform: scale(1); }
    15% { transform: scale(1.3); }
    30% { transform: scale(1); }
    45% { transform: scale(1.3); }
    60% { transform: scale(1); }
    100% { transform: scale(1); }
}

.heart {
    animation: heartbeat 1.2s infinite;
}

为什么使用CSS动画替代GIF？

更好的性能

CSS动画由浏览器GPU优化，比GIF占用更少的内存和CPU资源

更小的文件体积

使用雪碧图比完整GIF文件体积更小，加载更快

完全可控

可以轻松控制动画播放、暂停、速度、方向等

更高品质

CSS动画支持更高的帧率和分辨率，不会出现GIF的色彩限制
