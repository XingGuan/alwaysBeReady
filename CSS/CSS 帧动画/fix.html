雪碧图动画修复演示

此页面演示了如何正确实现5x5雪碧图动画。左侧是修复后的流畅动画，右侧是原始有问题的动画。

主要问题：原始代码使用了steps(24)，但5x5雪碧图实际上需要5步水平x5步垂直，共25帧。修复后使用了正确的步数并优化了关键帧。

原始问题动画

修复后的关键代码

.loading {
    width: 200px;
    height: 200px;
    background: url('sprite.png') 0 0 no-repeat;
    background-size: 500% 500%;
    animation: spriteAnimation 3s steps(5) infinite;
}

@keyframes spriteAnimation {
    0% { background-position: 0% 0%; }
    20% { background-position: 100% 0%; }
    40% { background-position: 0% 25%; }
    60% { background-position: 100% 25%; }
    80% { background-position: 0% 50%; }
    100% { background-position: 100% 50%; }
}
