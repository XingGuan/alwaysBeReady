### `React`的设计理念
+ 单向数据流
+ 虚拟`DOM`
+ 组件化  
  + 保持交互一致性  
  + 保持视觉风格的统一
  + 便于程序员互相之间的协作
> 组件化工作方式：信奉的就是独立，完整以及自由组合。   


#### 单向数据流  
+ 数据与界面绑定   
+ 单向渲染  
+ 就好像一个函数，同样的输入，同样的输出。
#### 虚拟`DOM`  
+ 类似`Docker`或`VMware`的`Snapshot`快照技术 
#### 组件化  
+ 保持交互一致性
+ 保持视觉风格统一
+ 便于程序员互相之间的协作
