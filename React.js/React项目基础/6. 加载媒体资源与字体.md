### `React`项目在引入`module.css`时项目会报错，但是引入`.svg`图片时确不会报错为什么？ 
#### `module.css`报错解决方案
`custom.d.ts`引入对象化的支持
```typescript
declare module "*.css" {
  const css: { [key: string]: string };
  export default css;
}

```
#### 引入`.svg`图片为什么不报错？  
因为`ReactTypeScript`在`react-app-tsts`中已经添加了对`svg`文件的支持
```typescript
declare module '*.svg' {
  import * as React from 'react';

  export const ReactComponent: React.FunctionComponent<React.SVGProps<
    SVGSVGElement
  > & { title?: string }>;

  const src: string;
  export default src;
}
```

### 添加字体  
首先需要下载字体  `ttf格式`
+ 引用`ttf`文件，因为字体是全局样式，所以需要在`index.css`文件中添加对字体的引用    
```css
 @font-face{
  font-family: 'Slidefu';
  src: local('Slidefu') ,url(./assets/fonts/Slidefu-Regular-2.ttf) format('truetype');
}
```  
使用字体文件方法  
```css
h1{
  font-family: 'Slidefu';
  font-size: 72px;
}
```
