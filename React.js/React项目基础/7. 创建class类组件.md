### 购物车(创建`class`类组件)    
`class component`  
[react生命周期](https://reactjs.org/docs/state-and-lifecycle.html)  

```javascript
class ShoppingCart extends React.Component{}
```
`React.Component`的参数
`class Component<P = {}, S = {}, SS = any>`
`P-props、S-state、SS-`自定义数据  
`SS`可以在`getSnapshotBeforeUpdate()`中使用，通过这个函数在`UI`渲染之前获取一些相应的`DOM`数据。  

给一个对象定义类型有两种方式，第一种使用关键词`type`,第二种使用`interface`
