### 配置开发环境  
> 脚手架工具 `creat-react-app` 
> 储备知识：终端或命令行、代码编辑器。

1.选择编辑器`VisualStudio Code`,简称`VSCode`。
2.安装`Node.js`。一般建议安装`LTS`版本，`LTS` 是 `“Long-Term Support”` 的缩写，中文意思是“长期支持”。
> 项目当前安装的`Node`版本是`18.1.0`  
3.搭建`React`项目  
    （1）脚手架工具 `create-react-app`
     > 拓展：查看包管理器镜像地址`npm get config registry`,`registry`注册表的意思。
     > 更换`npm`下载包管理器镜像为淘宝镜像地址`https://registry.npmmirror.com/`  
     执行`create-react-app`创建项目后，它都做了哪些事 
    创建`package.json`文件，其中`dependencies` 项目依赖，引用了我们项目中所有会使用到的第三方组件。   
    创建`public`文件夹，存放的是网站中的静态资源文件。
    创建`src`文件夹，就是项目中我们存放的**源代码**。
    > 拓展
    > 通过`npm`添加/安装新的dev项目依赖和(`devDependencies`)添加/安装新的项目依赖有什么区别？
    > 生产依赖（项目依赖）：`npm install package-name / npm install package-name --save` 这些是项目运行时必须的依赖
    > 开发依赖 (`devdenpendecies`)：`npm install package-name --save-dev / npm install package-name -D` 这些仅在开发阶段需要。
