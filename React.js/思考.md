### 7、`npx` 是什么？
>`npx`是`npm 5.2+` 附带的包运行器工具。`npx`可以在不全局安装工具集的情况下直接使用工具。

### 8、`npx` 使用淘宝镜像？  
+ 1.查看`npm`的镜像源  
```javascript
npm config get registry
``` 
+ 2.修改成淘宝的镜像源  
```javascript
npm config set registry https://registry.npm.taobao.org
```  
+ 3.然后执行`npx`命令进行安装
